# Project Template

## Folder structure

```
.
├───dist                   # Compiled files (alternatively `build`)
├───config                 # Project configuration files, such as database configuration, environment variable configuration, etc.
├───docs                   # Documentation files (alternatively `doc`)
│   ├───design
│   ├───development
│   ├───build
│   ├───testing
│   ├───deployment
│   ├───backup
│   ├───todo.md
│   └───usage.md           # Getting started guide
├───resources
│   └───images
├───scripts
├───solutions
├───src                    # Source files
│   ├───server
│   ├───app-android
│   ├───app-ios
│   ├───plugin
│   ├───web
│   ├───web-components
│   ├───web-hybrid
│   └───web-hybrid-components
├───tests
├───.gitignore
├───.markdownlint.yaml
├───.prettierrc.yaml
├───CHANGELOG.md
├───LICENSE
└───README.md
```
