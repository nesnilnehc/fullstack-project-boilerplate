# Project Template

## Folder structure

```
.
├───build                  # Compiled files (alternatively `dist`)
├───docs                   # Documentation files (alternatively `doc`)
│   ├── usage.md           # Getting started guide
│   └───todo.md
├───resources
│   └───images
├───scripts
├───solutions
├───src                    # Source files (alternatively `lib` or `app`)
│   ├───api
│   ├───app-android
│   ├───app-ios
│   ├───app-linux
│   ├───app-windows
│   ├───plugin
│   ├───web
│   ├───web-hybrid
│   ├───webcomponent
│   └───webcomponent-hybrid
├───.gitignore
├───.markdownlint.yaml
├───CHANGELOG.md
├───LICENSE
└───README.md
```
