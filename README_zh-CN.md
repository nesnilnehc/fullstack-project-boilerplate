# 项目模板

## 目录结构

```
.
├───dist                   # 编译后的文件（或者 `build`）
├───docs                   # 文档（或者 `doc`）
│   ├───design
│   ├───development
│   ├───build
│   ├───testing
│   ├───deployment
│   ├───backup
│   ├───TODO.md
│   └───USAGE.md           # 入门指南
├───config                 # 项目配置文件，如数据库配置、环境变量配置等
├───resources
│   └───images
├───scripts
├───src                    # 源码
│   ├───app-android
│   ├───app-ios
│   ├───server
│   ├───tools
│   ├───web
│   └───web-hybrid
├───tests
├───.gitignore
├───.gitlab-ci.yml
├───.markdownlint.yaml
├───.prettierrc.yaml
├───CHANGELOG.md
├───LICENSE
└───README.md
```
